## 项目介绍

快速搭建基于koa的后端项目模板，包含用户认证、路由、JSON数据库等基础功能。

使用到的库：

- koa 核心服务
- koa-body 解析请求体
- koa-router 路由
- koa-helmet 安全头
- koa-cors 跨域
- lowdb 本地JSON数据库
- jsonwebtoken 认证
- bcryptjs 密码加密
- dotenv 环境变量
- winston 日志
- koa-static 静态资源服务
- koa2-connect-history-api-fallback 处理单页面应用兼容
- pm2 进程管理
- nodemon 开发环境自动重启
